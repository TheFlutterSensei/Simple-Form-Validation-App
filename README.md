### Simple Form Validation App

Learn how to build forms that work.
This Flutter project shows how to validate user input.
Perfect for making real, usable apps.

Steps to use the app:
1. Clone the repository
2. Run ```flutter pub get```
3. Run ```flutter run```
4. Test form input and validation behavior
